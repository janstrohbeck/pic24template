void lcd_display_long_string ( c8_t *  pac8string,
uint16_t  u16delay 
)

Displays a longer string by splitting it at linebreaks and displaying the lines seperately.

The string is scrolling upwards every u16delay milliseconds.
It is also possible to use two linebreaks in sequence in the string to create an empty line on the display.

Parameters
pac8stringA pointer to the beginning of the string.
u16delayThe delay to be used between displaying the next two lines in milliseconds.
